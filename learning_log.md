## Day 1 - 2026-03-19 (环境突围与云端初始化)
### 1. 环境搭建与决策转变
- **起初尝试**：在本地使用 VMware 安装 Ubuntu 虚拟机，遇到网卡消失、无法修复及底层驱动冲突。
- **果断决策**：意识到虚拟机维护成本过高，果断卸载 VMware，转向使用 **腾讯云轻量应用服务器 (Ubuntu 22.04)**，显著提升了开发环境的稳定性。

### 2. 服务器连接与基础运维
- 学习使用 SSH 协议通过公网 IP (`119.45.153.130`) 远程管理服务器。
- 掌握了 Linux 基础目录操作 (`mkdir`, `cd`, `ls`) 以及编辑器 `nano` 的使用。

### 3. Git 与代码托管
- 配置了 Git 全局身份信息（用户名及邮箱）。
- 在服务器初始化 Git 仓库，并成功关联 GitHub 远程仓库。
- **安全实践**：遭遇了 GitHub Token 在命令行泄露的风险，并及时进行了 Token 撤销与重置。

## Day 2 - 2026-03-20 (后端起步：FastAPI 与公网服务)
### 1. Python 开发环境
- 安装并配置了 `python3-venv`，创建了项目独立的虚拟环境。
- 学习了为什么在服务器上要使用虚拟环境来隔离依赖。

### 2. FastAPI 接口开发
- 安装了 `fastapi` 和 `uvicorn`。
- 编写了第一个生产级后端接口 `main.py`，实现 JSON 数据返回。

### 3. 网络与安全配置
- **防火墙穿透**：通过腾讯云控制台手动配置防火墙规则，开放了 `8000` 端口的 TCP 访问。
- **跨设备访问**：成功在本地 Windows 浏览器通过 `http://119.45.153.130:8000/` 访问云端 API。
- **测试技巧**：初探了 FastAPI 自动生成的 `/docs` 交互式文档界面。
